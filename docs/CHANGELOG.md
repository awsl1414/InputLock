# Changelog

本文件记录 InputLock 的所有版本变更。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。

## [1.0.0] - 2026-04-26

首个正式版本。

### Added

- 输入法锁定 — 阻止 macOS 系统自动切换输入法（如窗口焦点变化时），同时放行用户快捷键主动切换
- 输入法选择 — 菜单子项列出系统已启用的输入法，选中项显示勾选标记
- 来源区分 — 100ms 时间窗口内通过快捷键检测（Ctrl+Space / Caps Lock）区分用户操作与系统自动切换
- 登录时启动 — 通过 SMAppService 注册为 macOS 登录项
- 首次启动引导 — 未授权时弹出引导窗口，授权完成后自动进入正常状态
- 菜单栏应用 — 无 Dock 图标，菜单栏图标实时反映锁定状态
- 关于窗口 — 显示应用名称、版本号和版权信息
